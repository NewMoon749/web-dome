# Carpeta Public

Coloca aquí los archivos estáticos de la aplicación.

## Archivos necesarios:

### Logo
- **`logo.png`** - Logo principal (usado en navbar)
- **`logo.ico`** - Favicon (usado en pestaña del navegador)

### Imágenes de fondo
- **`hero-background.jpg`** - Imagen de fondo para la sección Hero (recomendado: 1920x1080px o mayor)

**Recomendaciones para hero-background.jpg:**
- Formato: JPG o PNG
- Tamaño recomendado: 1920x1080px o mayor
- Tema: Imagen relacionada con abogados, justicia, o ambiente profesional legal
- La imagen se mostrará con efecto blur, así que puede ser una imagen más detallada
