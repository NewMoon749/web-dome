# EX LEGE-ON - Consultorio Jurídico Online

Página web del consultorio jurídico online EX LEGE-ON, diseñada para conectar a personas con abogados especializados de manera rápida y accesible.

## Características

- **Consultas Express**: Videollamadas de 30 min ($15) o 90 min ($35)
- **Servicios por Área**: Patrocinio legal especializado (Laboral, Familiar, Propiedad Intelectual) desde $150 hasta $600
- **Chatbot Inteligente**: Asistente virtual disponible 24/7 para guiar consultas
- **Sistema de Videollamada**: Opción de realizar ahora o programar, con selección de área y abogado disponible
- **Diseño Responsive**: Optimizado para todos los dispositivos

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React (iconos)

## Instalación

1. Instala las dependencias:
```bash
npm install
```

2. Ejecuta el servidor de desarrollo:
```bash
npm run dev
```

3. Abre [http://localhost:3000](http://localhost:3000) en tu navegador
